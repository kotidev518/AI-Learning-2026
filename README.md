# AI Learning 2026

This repository is my personal learning journey into Python, data analysis, and AI in 2026. It started as a simple practice workspace and has grown into a place where I explore small projects, notebooks, and datasets step by step.

## My learning journey

The journey began with basic Python building blocks and simple file operations. Over time, the workspace expanded into three main areas:

1. Building small Python programs in the projects folder.
2. Exploring data with Jupyter notebooks and pandas.
3. Practicing with real datasets to understand patterns, cleaning, and analysis.

Each folder represents a stage in that growth.

## Folder overview

- projects/: contains small Python scripts that demonstrate core programming ideas such as file handling, utilities, and basic program flow.
  - main.py shows a simple example of using the file processor.
  - processor.py provides helper methods for reading, writing, appending, and searching files.
  - utils.py contains reusable mathematical and string utility functions.

- notebooks/: holds Jupyter notebooks for experimentation and learning.
  - pandas_training.ipynb focuses on working with pandas and tabular data.
  - Other notebooks are used for exploratory practice and testing ideas.

- datasets/: stores sample data files used for analysis and practice.
  - The folder includes CSV files and dataset references for machine learning and data exploration exercises.

- sample.txt: a small sample file used by the project examples.

## What this repository represents

This workspace is not just a collection of files; it is a record of progress. It shows how a beginner can move from writing simple scripts to exploring data and preparing for more advanced AI work.

## Next steps

As the journey continues, I plan to add more notebooks, more data experiments, and eventually more advanced AI and machine learning projects.
